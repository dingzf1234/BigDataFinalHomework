# **大数据课程大作业**

本次大数据课后作业，我们小组选择的实习项目为《大作业的可能选题清单-20200521》文件中的第7项（Kaggle等），要求的实习任务如下：

- 数据来源：网上下载；
- 数据预处理：根据需要；
- 数据存储：HDFS、SQL/NoSQL等；
- 数据处理：机器学习、数据发掘等相关算法的Spark实现等；
- 数据可视化：自选（≥2种）。

**具体选题：**

基于大数据处理的房价评估模型。

**选题详细描述：**

首先，通过在Kaggle网站下载相应的房屋价格数据集。数据集中的数据类别包括租赁房源、小区、二手房、配套、新房、土地、人口、客户、房价等数据项。使用数据预处理和可视化方法处理数据集，并根据给定的数据集中每个房子的不同特征，采用大数据处理和机器学习的方法，基于Pyspark框架，建立模型，评估房子的价格。

**运行环境：**

- Hadoop: 2.6.4
- Spark: 2.4.5
- Python: 3.5.2
- IDE: Jupyter Notebook
